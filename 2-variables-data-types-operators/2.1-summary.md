JAVASCRIPT CHEAT SHEET (Practical)

1. Những kiểu dữ liệu bạn gặp nhiều nhất

string: Dùng cho văn bản, ví dụ tên đăng nhập, thông báo lỗi, tiêu đề trang.

number: Dùng cho số lượng, giá tiền, thời gian chờ, điểm số.

boolean: Chỉ có true hoặc false, rất hay dùng trong điều kiện.

undefined: Biến đã khai báo nhưng chưa có giá trị.

null: Bạn chủ động gán để nói rằng hiện tại không có dữ liệu.

object và array: Dùng khi dữ liệu có cấu trúc phức tạp hơn.

Ví dụ:
const user = { name: "Sharon", age: 25 }
const scores = [8, 9, 10]

2. Operators hay gặp

Arithmetic Operators (tính toán):
+  -  *  /  %  **

Comparison Operators (so sánh):
==  ===  !=  !==  >  <  >=  <=

Logical Operators (điều kiện):
&&   ||   !

Ternary Operator:
condition ? value1 : value2

Ví dụ:
age >= 18 ? "Adult" : "Child"

3. Các thao tác chuỗi hay dùng

trim():
Bỏ khoảng trắng thừa ở đầu và cuối chuỗi.
→ string

includes():
Kiểm tra chuỗi có chứa một đoạn text nào đó hay không.
→ boolean

replace() / replaceAll():
Thay thế ký tự hoặc đoạn text không mong muốn.
→ string

replace(/\D/g, ""):
Xoá hết các ký tự không phải số kể cả khoảng trắng.
Lưu ý: sai với số thập phân.

replace(/\s+/g, " "):
Gom nhiều khoảng trắng thành 1 khoảng trắng.

length:
Lấy độ dài chuỗi.
→ number

toLowerCase() / toUpperCase():
Chuẩn hóa chữ hoa / thường trước khi so sánh.
→ string

typeof:
Kiểm tra kiểu dữ liệu.

Mindset xử lý string:
LÀM SẠCH → TÌM MỐC → CẮT/TÁCH → CHUẨN HÓA → KIỂM TRA

--------------------------------------------------

4. Ép kiểu sang số trước khi tính toán

Number():
Đổi chuỗi sang số khi chuỗi đã khá “sạch”.
→ number

parseInt():
Lấy phần số nguyên.
→ number

parseFloat():
Lấy số thập phân.
→ number

Number.isNaN():
Kiểm tra kết quả ép kiểu có lỗi hay không.
→ boolean

toFixed():
Format số thập phân về số chữ số mong muốn.
→ string

Khi nào dùng gì:
- Number() → khi chuỗi phải là số sạch 100%
- parseInt()/parseFloat() → khi chuỗi vẫn còn dính text

5. Array methods hay dùng

push():
Thêm phần tử vào cuối mảng.

pop():
Xoá phần tử cuối mảng.

includes():
Kiểm tra giá trị có tồn tại không.

map():
Biến đổi toàn bộ mảng.

filter():
Lọc dữ liệu.

find():
Tìm phần tử đầu tiên thoả điều kiện.

reduce():
Gộp dữ liệu thành 1 giá trị.

Mindset xử lý array:
THÊM/XÓA → TÌM → LỌC → BIẾN ĐỔI → TỔNG HỢP

9. Core mindset khi code JS

Flow xử lý data thường là:

Input
→ Validate
→ Clean data
→ Convert data type
→ Process logic
→ Return output

Ví dụ thực tế:
User nhập số điện thoại
→ Xoá khoảng trắng
→ Xoá ký tự thừa
→ Chuẩn hóa format
→ Validate
→ Save
